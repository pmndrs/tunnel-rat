A potential cross renderer portal. It serves as a simple means to project something from one place to another.  Demo: https://codesandbox.io/s/basic-demo-forked-kxq8g

```jsx
import tunnel from 'tunnel-rat'

const t = tunnel()
// <t.In>{content}</t.In>
// <t.Out />
```
